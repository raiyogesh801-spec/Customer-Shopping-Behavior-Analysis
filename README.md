# Customer Shopping Behavior Analysis

## Overview

This project analyzes customer shopping behavior to understand purchasing patterns, product category performance, and customer demographics.

The analysis was conducted using Python (Pandas) for data processing and SQL querying, followed by visualization through a Power BI dashboard.

## Tools Used

* Python
* Pandas
* SQL
* Power BI

## Key Insights

* Clothing category generates the highest revenue.
* Young adults contribute the largest share of total sales.
* Majority of customers are non-subscribers.
* Average purchase amount is approximately $59.76.

## Dashboard

The insights were visualized using a Power BI dashboard displaying revenue distribution, sales trends, and customer behavior patterns.
